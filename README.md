# ClickClick — Video Marketing Solutions

One-pager for **ClickClick** — consultations and video marketing software tools for Live Commerce, Social Listening Signals, Influencer Marketing, TV advertising and Digital Signage.

**Contact:** hello@clickclick.video

## Preview locally

```bash
python3 -m http.server 5174
```

Open http://localhost:5174

## Publish on GitHub + your domain

1. Create a GitHub repo and push this folder.
2. Settings → Pages → Deploy from branch → `main` / root.
3. Add your custom domain and point DNS as GitHub instructs.
